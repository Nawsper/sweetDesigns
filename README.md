## Sweet Designs

Se utilizo para desarrollar este proyecto las siguientes tecnologías:
+ HTML
+ CSS

+ Metodología BEM
+ Git
+ Github/ Github Pages
+ SASS
  * Variables y mapas
  * Anidamientos (nesting y &)
  * Extend
  * Mixin e include
+ SEO
